Aldebarion

The front-end framework built for the future

Aldebarion is the front-end framework you need to design futuristic interfaces. First, it managed all disruptive screens like HUDs, smart boards, touch tables, etc... So the layout and the colors are take account of these screens. Then, it has a design built for productivity and long life: it is futuristic, refined, efficient and ergonomic. Finally, it introduces several level of reading to make interface adaptive to user and increase productivity again.

This project is just beginning so do not hesitate to propose improvement on github. For now, only a less/css implementation is available but I am working on a React.js implementation.
